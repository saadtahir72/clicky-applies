# Clicky - AI Applies

Track AI/ML job applications automatically and save to Google Sheets.

## Features
- One-click job tracking
- Auto-detects job info
- Google Sheets integration
- Daily/monthly dashboards

## Installation
1. Download from Chrome Web Store
2. Or load unpacked extension in Chrome

## Setup
1. Create Google Sheet
2. Deploy Apps Script
3. Configure Web App URL in extension settings
